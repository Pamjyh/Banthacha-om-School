# ระบบเก็บค่ารถ — คู่มือติดตั้ง

ระบบเก็บค่ารถนักเรียน โรงเรียนบ้านท่าชะอม
ใช้ Google Sheet **เล่มเดียวกับระบบออมทรัพย์** (อ่านรายชื่อจากชีต "นักเรียน") แต่ Code.gs และ Web App **แยกกัน** — ไม่กระทบระบบออมทรัพย์

---

## โครงสร้างข้อมูล

ใช้ Sheet เดิม (SHEET_ID เดียวกับออมทรัพย์) เพิ่ม 3 ชีตใหม่:

| ชีต | หน้าที่ | รีเซตตามเทอม? |
|-----|--------|--------------|
| `นักเรียน` | รายชื่อ (ของออมทรัพย์ อ่านอย่างเดียว) | — |
| `คนนั่งรถ` | ใครนั่งรถบ้าง | ❌ อยู่ถาวร |
| `ค่ารถ` | บันทึกจ่ายรายเดือน (1 แถว = 1 เดือนที่จ่าย) | ใช้ ปี+เทอม แยก |
| `บันทึกเงินนอก` | รายรับถาวร — **รีเซตเทอมแล้วไม่หาย** | ❌ อยู่ถาวร |

> ขึ้นเทอมใหม่ไม่ต้องลบอะไร — แค่เปลี่ยนปี/เทอมที่หัวจอ ระบบจะเริ่มนับใหม่ ของเก่ายังอยู่เป็นประวัติ

---

## ค่าค่ารถ (แก้ได้ใน Code.gs หัวไฟล์)

```js
const FARE_PER_MONTH = 100;
const TERM_MONTHS = {
  '1': ['มิ.ย.','ก.ค.','ส.ค.','ก.ย.'],   // เทอม 1 = 4 เดือน = 400
  '2': ['พ.ย.','ธ.ค.','ม.ค.','ก.พ.']    // เทอม 2 = 4 เดือน = 400
};
```

ถ้าโรงเรียนเก็บเดือนอื่น/จำนวนเดือนต่างจากนี้ แก้ตรงนี้ที่เดียว ตารางกับยอดรวมจะปรับตาม

## ขั้นตอน Deploy

> ⚠️ **สำคัญ: ต้องสร้างเป็นโปรเจกต์ Apps Script ใหม่ (standalone) — ห้ามเพิ่มไฟล์ในโปรเจกต์ออมทรัพย์**
> เพราะไฟล์ .gs ทุกไฟล์ในโปรเจกต์เดียวกันใช้ global ร่วมกัน จะชนกับ `SHEET_ID`, `doGet`, `checkRole` ฯลฯ ของออมทรัพย์
> โปรเจกต์แยกยังอ่าน Sheet เล่มเดียวกันได้ เพราะเปิดผ่าน `SHEET_ID`

### 1) สร้างโปรเจกต์ใหม่ + วาง Code.gs
1. เปิด **script.google.com** → กด **โปรเจกต์ใหม่ (New project)**
2. ลบโค้ดตัวอย่างในไฟล์ออก แล้ววางโค้ดทั้งหมดจาก `Code.gs`
3. ตั้งชื่อโปรเจกต์ว่า **ระบบค่ารถ** → กด 💾 บันทึก
4. เลือกฟังก์ชัน `initSheets` แล้วกด ▶ Run (อนุญาตสิทธิ์ครั้งแรก) → จะสร้าง 3 ชีตใหม่ในเล่มออมทรัพย์ให้

> ถ้าเผลอวางไว้ในโปรเจกต์ออมทรัพย์แล้วขึ้น error `Identifier 'SHEET_ID' has already been declared` → ลบไฟล์นั้นออก แล้วทำใหม่ตามข้อ 1

### 2) Deploy เป็น Web App
1. กด **Deploy > New deployment** → ประเภท **Web app**
2. ตั้งค่า: Execute as = **Me**, Who has access = **Anyone**
3. กด Deploy → **คัดลอก Web app URL**

### 3) ตั้งค่า index.html
เปิด `index.html` แก้บรรทัด:
```js
var API = 'PASTE_YOUR_WEB_APP_URL_HERE';   // ← วาง URL จากข้อ 2)
```

### 4) Host (เลือกอย่างใดอย่างหนึ่ง)
- **เปิดจากเครื่อง**: ดับเบิลคลิก `index.html` ได้เลย
- **GitHub Pages**: push ขึ้น repo (เช่น `School-CarFare`) แล้วเปิด `pamjyh.github.io/School-CarFare`

### 5) เพิ่มการ์ดใน School Portal (ถ้าต้องการ)
ใน `School Portal/index.html` เพิ่มการ์ดใหม่ ชี้ `data-url` ไปที่ลิงก์ระบบค่ารถ

## การเชื่อมเงินนอกกับระบบบริหารงาน
ปุ่ม "บันทึกเข้าเงินนอก" เก็บยอดถาวรในชีต `บันทึกเงินนอก` (ไม่หายเมื่อรีเซตเทอม)
จากนั้นนำตัวเลขไปลงบัญชีเงินนอกในระบบบริหารงาน (มีปุ่มลิงก์เปิดให้)

> หากต้องการให้ส่งยอดเข้า Supabase finance ของระบบบริหารงานอัตโนมัติ
> แจ้งได้ — ต้องต่อ API กับตาราง `finance_transactions` (หมวดเงินนอกงบประมาณ) เพิ่ม

## วิธีใช้งานประจำ
1. **ต้นปี/มีเด็กใหม่** → แท็บ "คนนั่งรถ" เปิดสวิตช์คนที่นั่งรถ
2. **เก็บเงินรายวัน** → แท็บ "เก็บเงิน" เลือกชั้น ติ๊กช่องเดือนที่จ่าย (หรือปุ่ม "จ่าย 400" ทั้งเทอม)
3. **ดูภาพรวม** → แท็บ "แดชบอร์ด" ดูว่าชั้นไหนเก็บได้เท่าไร ใครค้าง
4. **ปิดยอดเข้าบัญชี** → แท็บ "เงินนอก" กดบันทึก แล้วลงระบบบริหารงาน

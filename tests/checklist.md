# Manual Test Checklist — School Portal

ทดสอบทุกครั้งก่อน commit หรือแก้ไข index.html

## 1. หน้าหลัก (Page Load)
- [ ] เปิด `index.html` ใน browser โดยไม่มี server
- [ ] ชื่อโรงเรียน "โรงเรียนบ้านท่าชะอม" แสดงใน header
- [ ] ที่อยู่ "ต.เขากวางทอง อ.หนองฉาง จ.อุทัยธานี" แสดงใน footer ของ header
- [ ] เครดิต "Pamjyh_ & Claude by Anthropic" แสดงใน footer ล่างสุด

## 2. นาฬิกาและวันที่
- [ ] header แสดงเวลา HH:MM:SS อัพเดทเรียลไทม์
- [ ] header แสดง "วันที่ D เดือน YYYY" เป็นภาษาไทย (พ.ศ.) **ไม่ใช่ "วันที่ —"**
- [ ] section hero แสดงวันที่เหมือนกับ header
- [ ] ปีเป็น พ.ศ. (เช่น 2568, 2569) ไม่ใช่ ค.ศ.
- [ ] วันเป็นชื่อไทย เช่น "วันจันทร์", "วันศุกร์"
- [ ] เดือนเป็นชื่อย่อไทย เช่น "ม.ค.", "มิ.ย."

## 3. คำทักทาย
- [ ] 05:00–11:59: แสดง "สวัสดีตอนเช้า" พร้อม 🌅
- [ ] 12:00–12:59: แสดง "สวัสดีตอนเที่ยง" พร้อม ☀️
- [ ] 13:00–17:59: แสดง "สวัสดีตอนบ่าย" พร้อม 🌤
- [ ] 18:00–20:59: แสดง "สวัสดีตอนเย็น" พร้อม 🌆
- [ ] 21:00–04:59: แสดง "สวัสดีตอนดึก" พร้อม 🌙

  **วิธีทดสอบเร็ว**: แก้ `new Date()` ชั่วคราว เช่น `new Date('2025-01-01T06:00:00')`

## 4. Cards
- [ ] แสดง 4 cards: บริหารงาน, ออมทรัพย์, ลงเวลาทำงาน, สารบัญ
- [ ] card animation เล่น (fade-in + slide-up) เมื่อโหลดหน้า
- [ ] card "สารบัญ" มองเห็นได้ (ไม่หายไประหว่าง animation)
- [ ] card "สารบัญ" แสดง badge/label "เร็วๆ นี้" หรือสัญลักษณ์ inactive
- [ ] กด card "สารบัญ" แล้ว **ไม่มีอะไรเกิดขึ้น** (ไม่เปิด overlay)

## 5. การเปิดระบบ (iframe overlay)
ทดสอบแต่ละ card ที่ active:

### 5a. บริหารงาน
- [ ] กดแล้ว overlay เปิด ชื่อ "บริหารงาน" แสดงที่ top bar
- [ ] iframe โหลด https://pamjyh.github.io/School-Management-System/
- [ ] หน้า Portal (card grid) ถูกซ่อน ไม่มองเห็น

### 5b. ออมทรัพย์
- [ ] กดแล้ว overlay เปิด ชื่อ "ออมทรัพย์" แสดงที่ top bar
- [ ] iframe โหลด https://pamjyh.github.io/School-Savings/

### 5c. ลงเวลาทำงาน
- [ ] กดแล้ว overlay เปิด ชื่อ "ลงเวลาทำงาน" แสดงที่ top bar
- [ ] iframe โหลด `../ระบบเช็คชื่อมาทำงาน/index.html`

## 6. การกลับหน้าหลัก
- [ ] กดปุ่ม "← กลับหน้าหลัก" → overlay ปิด, หน้า Portal กลับมา
- [ ] กด Escape → overlay ปิด, หน้า Portal กลับมา
- [ ] หลังปิด overlay: iframe src เป็น `about:blank` (ไม่โหลดซ้ำ)
- [ ] หลังปิด overlay: focus กลับมาที่ card ที่กดเปิด

## 7. Keyboard Navigation
- [ ] Tab ผ่าน cards ได้ตามลำดับ มีกรอบ focus เห็นชัด
- [ ] กด Enter/Space บน card → เปิดระบบ
- [ ] เมื่อ overlay เปิด: focus อยู่ที่ปุ่ม "กลับหน้าหลัก"
- [ ] Tab ใน overlay: วนอยู่แค่ใน top bar (ไม่ไปที่ iframe content)
- [ ] Escape ปิด overlay ได้จากทุกที่

## 8. Logo
- [ ] โลโก้โรงเรียนแสดงใน header (ถ้ามีไฟล์รูป)
- [ ] ถ้าไม่มีไฟล์รูป: placeholder แสดงแทน (ไม่มี broken image icon)

## 9. Responsive (ถ้าจำเป็น)
- [ ] หน้าจอขนาด 1280px: cards แสดง 2×2 grid
- [ ] หน้าจอขนาด 768px: cards แสดงเป็น 1 คอลัมน์หรือ 2 คอลัมน์
- [ ] ไม่มี horizontal scroll

## 10. Console Errors
- [ ] เปิด DevTools → Console
- [ ] ไม่มี Error (สีแดง) เมื่อโหลดหน้า
- [ ] ไม่มี Error เมื่อเปิด/ปิดแต่ละระบบ
